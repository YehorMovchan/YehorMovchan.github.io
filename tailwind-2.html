Tailwind — приклади станів

Жива сторінка з прикладами поведінки класів зі станами (hover, focus, active, checked, peer). Код під кожним прикладом — копіюй і запускай.

Джерело: офіційна документація Tailwind — Hover, focus, and other states.

1) Hover / Focus / Active

Класичний приклад — кнопка з фоновим кольором для станів.

<button class="bg-sky-500 hover:bg-sky-700 focus:outline-2 focus:outline-offset-2 focus:outline-sky-600 active:bg-sky-800 text-white px-4 py-2 rounded">Наведи / Фокус / Актив</button>

2) Focus vs Focus-visible

focus-visible зазвичай показує індикатори лише для клавіатурної навігації (де це підтримується), тоді як focus реагує на будь-який фокус.

<input class="px-3 py-2 border rounded focus:outline-none focus:ring-2 focus:ring-sky-400" placeholder="focus..." />
<input class="px-3 py-2 border rounded focus:outline-none focus-visible:ring-4 focus-visible:ring-sky-600" placeholder="focus-visible..." />

3) Peer-checked (чекбокси)

Використовуємо peer на input і peer-checked: на нащадках для стилізації залежно від стану чекбоксу.

          <input type="checkbox" name="" id="example1" class="peer" />
            <label for="example1" class="peer-checked:text-red-300">Варіант 0</label>
          
          <label class="flex items-center gap-3">
  <input type="checkbox" class="peer hidden" />
  <span class="px-3 py-2 border rounded peer-checked:bg-sky-600 peer-checked:text-white">Варіант 1</span>
</label>

4) Peer-checked (радіокнопки)

Радіокнопки дозволяють вибрати лише один варіант. Використовуємо peer-checked: для стилізації сусідніх елементів.

<label>
  <input type="radio" name="option" class="peer hidden" />
  <div class="px-4 py-2 border rounded peer-checked:bg-purple-600 peer-checked:text-white">
    Варіант A
    <p class="text-xs text-gray-500 peer-checked:text-purple-100">Додатковий опис</p>
  </div>
</label>

5) Odd / Even — почергові рядки

Корисно для таблиць або списків — odd: і even: змінюють стилі по черзі.

  • Рядок 1
  • Рядок 2
  • Рядок 3
  • Рядок 4
<ul class="p-3 odd:bg-white even:bg-gray-50">...</li>

Стилізація нащадків через * та **

Можна стилізувати всіх дітей (*:) або всіх нащадків будь-якого рівня (**:):

  • Прямий нащадок div
  • Прямий нащадок div
<div class="border p-4 rounded space-y-2">
  <ul class="*:text-blue-600 *:italic">
    <li>Прямий нащадок div</li>
    <li>Прямий нащадок div</li>
  </ul>
</div>
